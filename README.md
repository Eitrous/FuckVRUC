# 微人大 ++
现在的微人大同时拥有着极其难看的 UI 和极其难用的操作逻辑，新推出的数智人大更是一坨。如果你也受不了以下槽点：

- 莫名其妙的登味审美风格 UI
- 常用的功能永远无法第一时间找到
- 期末周查成绩每次都要被“温馨提示”硬控几秒

欢迎来尝试「微人大 ++」

## 这是什么

这是一个浏览器插件，致力于恢复门户网站该有的便利与人文关怀。

目前已有功能：

- [x] 常用链接聚合
- [x] 一键查询成绩
- [x] 一键查询课表
- [x] 简化数智人大

todo:
- [ ] 添加更多常用链接
- [ ] ……

> :warning: 本项目**不会**收集个人信息，所有查询工作均通过调用微人大和教务现有的接口完成

> :warning: 在没有连接到校园网时需要使用校内 VPN ，否则会显示 `Failed to fetch`

## 我该如何使用

1. 前往 [Releases](https://github.com/Eitrous/FuckVRUC/releases) 页下载插件
2. 给你的浏览器安装这个插件。请自行搜索「Chrome插件安装方法」
